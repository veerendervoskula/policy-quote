# Policy Quote Application

## Overview
Full-stack minimal home insurance quoting tool demonstrating Angular 17 + Node.js architecture.

## Tech Stack
- Angular 17 (Standalone + Signals)
- Node.js (Express)
- TypeScript (strict)
- Canvas API (chart)


## Features
- Reactive form with validation
- Risk-based underwriting logic
- Premium calculation
- Risk band visualization
- Error handling + logging
- Accessible UI (WCAG)

## Run Locally
npm start to kick start frontend and backend


### Backend
cd packages/backend
npm install
npm run dev
npm run test

### Frontend
cd backend/frontend
npm install
ng serve
npm run test
